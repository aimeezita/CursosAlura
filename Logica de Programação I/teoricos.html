Exercícios Teóricos do Curso de Lógica de programação da Alura

Onde está o erro?

Sílvia escreveu o seguinte código:

<meta charset="UTF-8">
<script>
    var minhaIdade = 20;

    document.write("Eu nasci em " + (2016 - minhaidade) );
</script>

O código de Sílvia não funciona. Qual é o problema?

Resposta: A variável minhaIdade está escrita de duas formas diferentes, uma com Idade com letra maíuscula, outra com letra minúscula, dessa forma é como se fossem duas variáveis diferentes, mas a que está no código não está declarada.
